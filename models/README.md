# Models directory

This folder contains trained AI models for the browser.

## Structure

```
models/
├── intent-classifier/                    # Browser-ready model (USE THIS)
│   ├── model.safetensors                 # 128MB FP16 weights
│   ├── config.json                       # Model config with labels
│   ├── vocab.txt                         # Vocabulary
│   ├── tokenizer.json                    # Tokenizer
│   ├── tokenizer_config.json             # Tokenizer config
│   └── special_tokens_map.json           # Special tokens
│
├── distilbert-navigation-quantized/      # Quantized source model
│   └── ...
│
└── distilbert-navigation-finetuned/      # Original trained model (FP32)
    └── ...
```

## Usage

The browser automatically loads from `intent-classifier/` via the embedded HTTP server on port 3737.

## Generated By

```bash
cd ../model-training
python3 prepare_model_for_browser.py
```

## Size

- `intent-classifier/`: ~128 MB (FP16)
- `distilbert-navigation-quantized/`: ~128 MB (FP16)
- `distilbert-navigation-finetuned/`: ~255 MB (FP32)
